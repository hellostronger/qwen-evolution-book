# 第6章 Qwen 3 系列：全新架构时代（2026年）

> 本章要点：Qwen 3系列的MoE架构创新、多尺寸版本矩阵与性能突破。

- **发布时间**：2026年初
- **定位**：全新一代架构，引入MoE混合专家

## 6.2 Qwen 3 系列版本矩阵

### 稠密模型

| 模型 | 参数量 | 定位 |
|-----|-------|------|
| **Qwen3-1.7B-Base** | 1.7B | 轻量级基础模型 |

### 混合专家模型

| 模型 | 总参数 | 活跃参数 | 定位 |
|-----|-------|---------|------|
| **Qwen3-30B-A3B-Base** | 30B | 3B | MoE高效推理 |

**MoE架构说明**：

```
MoE (Mixture of Experts) 原理：

┌────────────────────────────────────────────────────┐
│              MoE 架构示意                           │
├────────────────────────────────────────────────────┤
│                                                    │
│  传统Dense:                                        │
│  输入 → 所有参数 → 输出                            │
│  (30B参数全部参与计算)                             │
│                                                    │
│  MoE:                                              │
│  输入 → Router(路由) → 选择Top-K专家 → 输出         │
│                                                    │
│  ┌─────────────┐                                  │
│  │ Expert 1    │ ─┐                               │
│  │ Expert 2    │ ─┼→ 每次只激活3B参数              │
│  │ Expert 3    │ ─┤                               │
│  │ ...         │ ─┼→ Router动态选择               │
│  │ Expert N    │ ─┘                               │
│  └─────────────┘                                  │
│                                                    │
│  效果：                                            │
│  - 推理成本 ≈ 3B稠密模型                           │
│  - 性能 ≈ 30B稠密模型                              │
│  - 效率提升约10倍                                  │
│                                                    │
└────────────────────────────────────────────────────┘
```

## 6.3 Qwen 3.5 系列：成熟优化

### 版本矩阵

| 模型 | 参数量 | 类型 | 定位 |
|-----|-------|------|------|
| **Qwen3.5-2B-Base** | 2B | 稠密 | 轻量级基础 |
| **Qwen3.5-9B-Base** | 9B | 稠密 | 中型基础 |
| **Qwen3.5-27B** | 27B | 指令 | 大型指令模型 |
| **Qwen3.5-35B-A3B** | 35B总/3B活 | MoE | 高效旗舰 |

### 关键改进

从 Qwen 3 → Qwen 3.5：

| 改进点 | 说明 |
|-------|------|
| **模型规模扩展** | 增加更多尺寸选择 |
| **指令模型强化** | Qwen3.5-27B 专为对话优化 |
| **MoE效率优化** | 35B-A3B 更大容量，保持高效 |
| **多模态基础** | 为后续3.7-Plus做准备 |
